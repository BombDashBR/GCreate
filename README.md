# GCreate
A program to facilitate the creation of mods for Garry's Mod.

Hello, I'm creating a program in Python where you create mods for gmod. To create a mod in Garry's Mod, you need to create a folder and remember the paths to the mods (Lua, materials, etc.). To create weapons, you have to go to forums to find the basic script to create the weapon. That's why I created GCreator, which makes creating mods easier. However, you will still need to learn gmod lua, because the application (still) doesn't have a language that facilitates mod creation, like mcreator, which allows you to create mods without writing a single line of code. GCreator is more useful for creating weapons, in case you don't want to program.
